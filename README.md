# Finding Minimum Of F(x) Using Multiple Methods

$$F(x)=x^6-2x^5+5x^4+50x^3+15x^2-20x+100$$

## Methods

- Gradient Descent (LMS)
- Gradient Descent with Momentum
- Conjugate Gradient
- Levenberg-Marquardt

## requirements

- Python 3.7 or above
- Jupyter Notebook

## libraries

- numpy
- matplotlib
